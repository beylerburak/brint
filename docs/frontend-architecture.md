# Frontend Architecture (Skeleton)

> Bu doküman Next.js + i18n + ShadCN + feature-based mimariyi tanımlar. Şu an **skeleton** durumdadır.

Başlıca başlıklar:
- Next.js App Router + route stratejisi (locale + workspace-first + brand studio)
- i18n yapısı (tr/en)
- ShadCN (New York style, accent, dark mode)
- Feature-based klasör yapısı
- HTTP client & error handling
- Forms (React Hook Form + Zod)
- Toast & global error boundary

Tiny Step ilerledikçe bu doküman güncellenecektir.
