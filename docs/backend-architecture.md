# Backend Architecture (Skeleton)

> Bu doküman backend tarafındaki mimari kararları tanımlar. Şu an **skeleton** durumdadır ve adım adım doldurulacaktır.

Başlıca başlıklar:
- Katmanlı yapı (core, modules, domain, application, infrastructure, presentation)
- Config ve env yönetimi
- Logger ve global error handler
- Redis & cache abstractions
- Auth & Claims altyapısı
- Endpoint ekleme rehberi

Tiny Step ilerledikçe bu doküman güncellenecektir.
